Este projeto é o nosso PIM - Projeto Integrado Multidisciplinar que estamos desenvolvendo em 7 pessoas, contendo a parte prática (código) e a parte documental (word em formato ABNT de trabalhos acadêmicos).

Estamos criando um site para venda de ingressos para a Copa Do Mundo de 2026, que por enquanto é ainda apenas um protótipo, já com a programação em C# básico (apenas as classes) e Banco De Dados, mas ainda
sem grandes funcionalidades e ainda sem API linkando o código C# ao Banco de Dados.
